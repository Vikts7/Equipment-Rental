# Equipment-Rental

Държавен изпит 2026 ИТ Кариера Виктория Цачева

Системата „Equipment Rental“ е уеб приложение за управление на оборудване и заявки за неговото използване. Тя позволява на потребители да разглеждат налично оборудване и да подават заявки за използване на един или повече артикули за определен период от време.

# Използвани технологии:

Frontend: React + Vite
Backend: Node.js
Database: PostgreSQL pgAdmin4
ORM: Sequelize
Nodemailer
JWT

Системата поддържа два типа потребители:
Първи регистриран потрбител получава роля админ
